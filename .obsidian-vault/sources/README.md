# Raw sources

`sources/` is the evidence layer. Store URLs, captured excerpts, repository audit notes, screenshots, or dated exports here when they are needed to explain a decision.

Raw source pages are immutable. Do not rewrite an old capture to reflect later understanding; add a new dated page and update the wiki pages that interpret it. Keep source notes concise and cite the original URL or repository path.

The initial external source is [[sources/karpathy-second-brain]]. The gist is referenced rather than duplicated because the useful ideas are structural and the URL is stable. Future ingestion should record retrieval date, scope, and any limitations.
